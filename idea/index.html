Ayuda

(Especialmente para el chico Juan si no recuerdas esta parte)

Es muy fácil editar esta página:

  1. Crear una carpeta en su computador donde vayan a clonar el repositorio.
  2. Dentro de la carpeta, abrir la consola de git (Git Bash) y Clonar el repositorio con el comando git clone https://github.com/maopinedaangel/pipa.git
  3. Abrir la carpeta clonada con Visual Studio Code.
  4. ¡Y empezar a editar! No se necesita instalar nada.
  5. Para subir los cambios, ubicarse en la carpeta del repositorio y abrir la consola de Git (Git Bash)
  6. Escribir el comando git add . para agregar a seguimiento todos los archivos modificados.
  7. Escribir el comando git commit -m "Mensaje" para confirmar los archivos.
  8. Escribir el comando git push origin main para subir los cambios al repositorio remoto (a Github). al realizar esta acción, es posible que Git solicite las credenciales, es decir, nombre de uusario y contraseña.
